
# Next.js 13 .tsx 打印网页按钮

```ts
import React from "react";

const PrintButton = () => {
  const handlePrint = () => {
    window.print();
  };

  return (
    <button onClick={handlePrint}>打印网页</button>
  );
};

export default PrintButton;
```

## 只想打印 <div class="book"></div> 里的内容
```css
@media print {
  /* 隐藏 body 元素以外的所有内容 */
  body *:not(.book) {
    display: none;
  }

  /* 为打印提供样式 */
  .book {
    margin: 0;
    padding: 0;
    font-size: 16px;
    line-height: 1.5;
  }

  /* 移除页面边距和页眉页脚 */
  @page {
    margin: 0;
  }

  /* 添加页码页眉等信息 */
  @page :first {
    /* 在第一页的顶部添加书名和作者 */
    @top-left {
      content: "书名：《XXXXX》";
    }
    @top-right {
      content: "作者：XXXXX";
    }
  }
}
```
